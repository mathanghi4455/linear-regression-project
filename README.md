# linear-regression-project
This project demonstrates a basic Linear Regression model implemented using Python. It shows how machine learning can be used to understand the relationship between input and output data and make predictions.

The model is trained on a small dataset of input values (X) and corresponding outputs (y). After training, it is capable of predicting results for new input values provided by the user.

This project is designed as a beginner-friendly example to understand the fundamentals of machine learning and prediction.
